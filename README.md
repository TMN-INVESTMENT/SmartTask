# SmartTask

SmartTask is a revolutionary task-based earning platform that empowers users to generate income by completing simple daily tasks and investing in VIP packages. Built with cutting-edge technology and a user-centric design, SmartTask provides a seamless experience for earning, investing, and growing your financial portfolio.

🎯 Mission Statement

"To create a sustainable ecosystem where every user can earn through simple tasks, grow through smart investments, and build wealth through community referrals."

💡 Core Concept

SmartTask combines three powerful earning mechanisms into one unified platform:

1. 📋 Task-Based Earnings

Users earn daily profits by completing simple rating tasks. Each day, users receive 3 tasks that require rating products, watching videos, or completing surveys. Upon completing all tasks, users unlock their daily profit from active VIP packages.

2. 💼 Investment Opportunities

SmartTask offers 8 VIP investment packages ranging from 10,000 TZS to 1,000,000 TZS. Each package generates daily profits between 500 TZS to 60,000 TZS, with monthly returns up to 180%.

3. 👥 Referral Network

A powerful 3-level referral system allows users to earn commissions from their referrals:

· Level 1: 10% commission
· Level 2: 7% commission
· Level 3: 3% commission
